# PlayerOK HUB

**PlayerOK HUB** — Telegram-сервис для автоматизации продаж на PlayerOK.

Сервис помогает продавцам быстрее работать с товарами, заказами, чатами, автоответами, восстановлением карточек, автоподнятием и учётом прибыли.

## Что делает сервис

PlayerOK HUB создан для продавцов, которые хотят меньше выполнять ручных действий и больше управлять продажами через Telegram.

Основные возможности:

* автовосстановление проданных товаров;
* автоподнятие карточек в каталоге;
* автоответчик для покупателей;
* уведомления о заказах и сообщениях;
* работа с чатами PlayerOK;
* бухгалтерия и учёт прибыли;
* статистика продаж;
* контроль ошибок;
* работа с прокси;
* управление модулями через Telegram-бота.

## Для кого подходит PlayerOK HUB

Сервис подойдёт продавцам PlayerOK, которые:

* продают много товаров;
* хотят быстрее отвечать покупателям;
* не хотят вручную восстанавливать проданные карточки;
* хотят держать товары выше в каталоге;
* хотят получать уведомления в Telegram;
* хотят считать расходы, комиссии и чистую прибыль;
* хотят масштабировать продажи.

## Основные модули

### Автовосстановление товаров

Если товар купили, сервис может автоматически восстановить карточку из завершённых, чтобы товар снова появился в продаже.

Можно настроить:

* восстановление по конкретному товару;
* восстановление по группе товаров;
* количество активных карточек;
* точное или похожее совпадение;
* повторные проверки, если товар временно не найден.

### Автоподнятие

Сервис проверяет позицию товара в каталоге PlayerOK и поднимает карточку, если она опустилась ниже выбранной позиции.

Если товар уже находится выше целевой позиции, поднятие не тратится.

### Автоответчик

Автоответчик помогает автоматически отправлять сообщения покупателям.

Можно настроить:

* приветствие после покупки;
* инструкцию покупателю;
* ответы по ключевым словам;
* разные сценарии для разных товаров;
* сообщения для групп товаров.

### Уведомления

Бот отправляет уведомления о важных событиях:

* новый заказ;
* новое сообщение;
* товар восстановлен;
* товар не найден;
* автоподнятие выполнено;
* ошибка прокси;
* проблема с аккаунтом;
* важное событие по товару.

### Чаты

Раздел чатов помогает продавцу быстрее работать с покупателями и видеть, по какому товару идёт диалог.

### Бухгалтерия

Бухгалтерия помогает учитывать:

* продажи;
* расходы;
* комиссии;
* чистую прибыль;
* прогноз прибыли;
* кассовый разрыв;
* отчёты.

## Как это работает

1. Пользователь подключает аккаунт PlayerOK.
2. В Telegram-боте выбирает нужные модули.
3. Настраивает товары, автоответы, автоподнятие и уведомления.
4. Сервис на сервере проверяет товары, заказы, позиции и сообщения.
5. При нужном событии сервис выполняет действие.
6. Пользователь получает результат и уведомления в Telegram.

## Статус проекта
